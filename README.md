# 📝 Simple Text Editor (Python Tkinter)

A simple GUI-based text editor built using **Python** and the **Tkinter** library. This application allows users to create, open, edit, and save `.txt` files through an easy-to-use graphical interface.

## 🚀 Features

- Create a **new file**
- Open existing **`.txt` files**
- Edit text in a **text area**
- **Save** and **Save As** functionality
- Simple **File menu**
